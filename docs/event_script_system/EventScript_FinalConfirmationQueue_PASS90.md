# EventScript Final Confirmation Queue - Pass 90

Pass 90 introduces a review model for final exact naming.

## Buckets

- `confirmed_direct_anchor`: the entry has enough direct text/anchor evidence and is closed for this pass.
- `domain_specific_needs_exact_scene_name`: the entry has a useful domain owner, but exact scene naming is still optional/manual.
- `structural_lane_needs_exact_owner_or_scene`: the entry has a stable structural alias, but the exact owner, NPC, cutscene, or scene name still requires confirmation.

- `reports/pass90_eventscript_final_confirmation_matrix.csv`
- `reports/pass90_final_confirmation_unit_queue.csv`
- `reports/pass90_final_confirmation_summary.csv`
- `reports/pass90_remaining_exact_name_area_percentages.csv`
- `reports/pass90_confirmation_lane_summary.csv`
