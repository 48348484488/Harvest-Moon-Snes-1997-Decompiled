## What this package is

- A NO-ROM decompilation/rebuild workspace for Harvest Moon SNES USA.
- ASM source and data banks prepared for byte-perfect rebuild when the user supplies a clean original ROM locally.
- Documentation and reports from the decompilation passes.
- Tools used to audit, classify and validate EventScript, GOBJ/visual references and semantic naming.

## What this package is not

- It does not include a ROM.
- It does not include a generated `.sfc` or `.smc` build artifact.
- It is not a commercial game distribution.
