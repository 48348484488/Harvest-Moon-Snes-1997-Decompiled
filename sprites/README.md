# sprites/

Pasta reservada para os atlases de sprite extraídos da ROM por banco
(`bank_88.png` ... `bank_91.png`, 10 arquivos).

Essas imagens foram removidas da publicação inicial deste repositório por serem arte original
do jogo (ver `DISCLAIMER.md` na raiz). Se você tem uma cópia legal da ROM e quiser subir essas
imagens aqui mesmo assim, gere-as com `tools/BitplaneToPng.py` / `tools/SpriteDecompressor.py`.
